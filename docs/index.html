This page contains tutorials and the documentation for Cacatoo, a javascript library to build simple IBMs. The simulation you see below is fully interactive, making it much easier to play with and try different parameters. But perhaps most importantly, it is super easy to make yourself! To get started, use the menu on the right-hand side to navigate through the manual.
